## How the mod works
fractal terrain uses a deep learning model to generate its terrain. 
Don´t worry, the model was trained using Mother Nature's beautyfull landscapes captured in the form of DEMs via satelite data and other means. Most importantly, not a single image is stolen from an artist and all data is permited for use.
# Instalation
Make sure you run this mod with java 21, (even though 1.20.1 can run on java 17) the recommended java instalation that modrinth provides is good.

Once you downloaded the mod, you will also have to download the models' weights via a datapack in this link: [download datapack](https://huggingface.co/batata1-jg/fractalterrainweights/blob/main/default_weights-1.0-alpha.zip)

Note: the file is large (1GB) because the network has a lot of parameters
Another note: the mod will not work without this datapack
just drag and drop this into the datapacks folder before creating your world and you are good to go!
## Features
This mod has two settings, one for large terrain and another for smaller scale terrain.
- The large generates terrain up to 1800 blocks in height 
- The small one is a scaled down version that generates up to 400 blocks in height
In this mod, you can experience terrain that seems eroded in a more realistic manner, but also more creative types of landforms.
